Projects

AI-Powered Chatbot Web App

Built a sleek and responsive chatbot interface using Next.js, ShadCN, and integrated Gemini API to handle user prompts and generate intelligent responses. The app features a smooth, modern UI and supports multi-turn conversations, image input, and basic document upload functionalities—offering a real-time, interactive AI assistant experience right in the browser.

Calorie Counter App

Uses AI to detect food and calculate nutrition from an image. Built with HTML, CSS, JS & llama-4-scout & ImageBB API.

3D Heart Reconstruction Using Ultrasonic Sensor Data

This project focuses on reconstructing a 3D model of the human heart using time-intensity data from ultrasonic sensors, diverging from traditional ultrasonography image-based methods.If results are achieved then it will work by interpreting limited 2D cross-sections and depth cues, the goal is to generate a detailed volumetric representation for improved cardiac analysis. Currently under active development, the project continues to evolve through ongoing experimentation.
